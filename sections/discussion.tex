\section{Deployment}
Er cloud deployment den optimale løsning for vores system? Hvad kunne man ellers have gjort?
\\\\
The choice to deploy the system in a cloud, is based on ethical considerations. If the system is incorporated directly into the local hardware, it would be implied that the captured footage is constantly being analyzed in real time. Such a design effectively normalizes monitoring the citizens at all times, which aligns with the definition of a surveillance state. Furthermore, it becomes more difficult to regulate and monitor how and whom uses the system, when it is embedded in hardware, which increases the risk of misuse. 
\\\\
On the contrary, if the system is deployed in the cloud, it remains inactive until invoked for a specific purpose. This ensures that footage is not automatically processed unless it is necessary and that access can be controlled via authentication and logs.
