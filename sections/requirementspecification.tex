\chapter{Requirement Specification} 
\label{cha:reqspec}

As defined in the problem statement, this project focuses on building and testing how \ac{SR} can be combined with and improve the performance of \ac{ReID}. Consequently, the requirements will focus on these two networks and their integration. For that same reason, the requirements will not describe a complete \ac{ReID} system and are therefore not exhaustive. The scope implies that the \ac{SR} and {ReID} integration does not include an object detection module, and thus expecting cropped images as input.
\\\\
The priority of the requirements are phrased using MoSCoW's prioritization method \cite{Moscow}. Each requirement will be testable and measurable with defined success criteria, and have a traceability tag.
